# Stage 3O evidence pack

Metadata-only. The deterministic artifacts are regenerated by
`tools/simurgh-benchmark/simurgh-benchmark.mjs`; the attestation is signed locally
(CI verifies only). No raw target output, no transcripts, no secrets.

| File | Contents |
|---|---|
| `corpus-manifest.json` | 150-case canary corpus summary + markers hash. |
| `reference-target-results.json` | Clean reference target metrics (120/120 contained, 0 over-defence). |
| `self-proof-results.json` | Adversarial detector counts (liar/leaker/overdefender/invalid). |
| `scoring-matrix-results.json` | 11 explicit fixtures proving every scoring cell. |
| `containment-attestation.json` | Signed run-set attestation (`simurgh.byo.attestation.v1`). |
| `containment-attestation.signature.json` | Ed25519 detached signature sidecar. |
| `attestation.public-key.json` | Dedicated Stage 3O public key (distinct from 3M). |
| `evidence-hashes.json` | sha256 of the full committed pack. |
| `generated-evidence-privacy-report.json` | Forbidden-token findings (must be 0). |
| `runner-output.txt` | Generation pass marker. |

Maintainer regeneration (local; needs the private key in `~/.simurgh/`):

```
node tools/simurgh-benchmark/simurgh-benchmark.mjs evidence --update
SIMURGH_BYO_PRIVATE_KEY_PATH=~/.simurgh/byo-ed25519.pem node tools/simurgh-benchmark/sign-byo-attestation.mjs
node tools/simurgh-benchmark/simurgh-benchmark.mjs hash
```

CI / reviewer (verify-only): `bash scripts/smoke-llm-shield-stage3o.sh`.

Run against your own gateway:

```
node tools/simurgh-benchmark/simurgh-benchmark.mjs run --target https://your-gateway/run --out result.json
```

External targets are reported as `measured_not_certified`.
