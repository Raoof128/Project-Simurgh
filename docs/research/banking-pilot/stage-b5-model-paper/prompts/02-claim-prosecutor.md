# Pass 2 — Claim prosecutor

Act as a hostile claim auditor for a privacy/security systems paper.

Review the provided text and identify every phrase that could be read as:
fraud detection, scam prevention, payment safety, payee verification,
financial advice, compliance, real banking protection, or production
readiness.

For each issue:

1. Quote the risky phrase.
2. Explain why it is risky.
3. Rewrite it safely.
4. Mark severity: low / medium / high.

Use only the evidence provided.
