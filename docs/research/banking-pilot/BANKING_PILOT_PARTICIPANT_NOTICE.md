# Banking Shield Participant Notice

## Phase A Status

Phase A has no human participants. This notice is a roadmap artifact for continuity.

## Plain-Language Description

Banking Shield is a banking-adjacent research demo using fictional scenarios only. It does not connect to a bank, process a payment, access an account, provide financial advice, detect fraud, or affect a real financial decision.

## Synthetic Activities

The Phase A demo presents fixed fictional categories for:

- Mock consent.
- Example payee-check category.
- Remote-access warning.
- Payment pause.
- Synthetic finance-agent approval.

## Data Not Requested

Participants must not enter credentials, OTPs, account numbers, balances, payees, transaction amounts, payment references, personal finance details, screenshots, app names, process names, or window titles.

## Withdrawal

The Phase A route supports withdrawal for the synthetic session. Withdrawn sessions cannot export reports.

## Future Use

Phase B or Phase C would require separate approval, updated notice text, and governance review before any human participation.
